## Django based class demo about Security essentials required by dev

## 📁 Project Structure

### Taremwa User Authentication Service (UAS)
A complete Django authentication application implementing secure user registration, login, and profile management.

**Quick Start**: See [TAREMWA_QUICKSTART.md](TAREMWA_QUICKSTART.md)  
**Full Documentation**: See [TAREMWA_README.md](TAREMWA_README.md)

#### Features
- ✅ User registration with validation
- ✅ Secure login/logout
- ✅ Protected dashboard
- ✅ Profile management
- ✅ Password change
- ✅ Django admin integration
- ✅ 20 comprehensive tests

#### Test the Application
```bash
# Run test suite
python manage.py test taremwa

# Start development server
python manage.py runserver

# Access at http://localhost:8000/auth/
```
